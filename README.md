# My Learning and Project Monorepo (2026 onwards)

This monorepo tracks my journey from a junior developer to a senior AI/MLOps engineer. It's a collection of my learning notes, practice projects, and experiments.

## Folder Structure

-   `apps/`: Contains practice projects and applications.
-   `packages/`: Contains learning notes, reusable components, and libraries.
-   `docs/`: Contains documentation.
-   `scripts/`: Contains scripts for automation.

## Usage

1.  **Install dependencies:**
    ```bash
    pnpm install
    ```

2.  **Run development server:**
    ```bash
    pnpm dev
    ```

3.  **Build all projects:**
    ```bash
    pnpm build
    ```
