### **Week 1: 语言本体（纯JS，不碰DOM）**

| 天数 | 主题 | 核心概念 | 当日挑战（必须提交代码） |
|------|------|----------|------------------------|
| **Day 1** | **变量与类型** | `let/const/var`、8种数据类型、类型转换陷阱 | 写一个函数`typeChecker(value)`，能准确识别`null`、`Array`、`Date`等边缘类型 |
| **Day 2** | **运算符与比较** | `==` vs `===`、`&&`短路、`??`空值合并 | 实现一个`safeGet(obj, path)`，用`?.`和`??`安全取值 |
| **Day 3** | **字符串模板** | 模板字面量、标签函数、`String.raw` | 写一个标签函数`sql`，防止SQL注入（转义单引号） |
| **Day 4** | **数字与BigInt** | 浮点精度、`Number.isFinite`、`BigInt` | 精确计算`0.1 + 0.2 === 0.3`（用BigInt或乘数法） |
| **Day 5** | **控制流** | if/else、switch、三元表达式、真值表 | 重构10层嵌套if-else为早期返回+策略对象 |
| **Day 6** | **循环与迭代** | `for`/`for...of`/`for...in`/`Array.forEach` | 用`for...of`遍历字符串中的Unicode字符（含emoji） |
| **Day 7** | **周复盘** | 整理Week 1笔记，刷5道CodeWars 8kyu题 | 提交Week 1总结到GitHub，至少1000字 |

### **Week 2: 函数与作用域**

| 天数 | 主题 | 核心概念 | 当日挑战 |
|------|------|----------|----------|
| **Day 8** | **函数声明与表达式** | 函数提升、一等公民、IIFE | 写一个`once(fn)`函数，让任何函数只能执行一次 |
| **Day 9** | **箭头函数** | 词法`this`、不能`new`、隐含return | 用一行箭头函数实现`compose(...fns)`（函数式编程） |
| **Day 10** | **作用域与闭包** | 词法作用域、闭包内存泄漏、模块化 | 实现`createCounter()`，每次调用+1，但不能用全局变量 |
| **Day 11** | **参数与rest** | 默认参数、`arguments`对象、`...rest` | 写一个`sum()`函数，支持`sum(1,2,3)`和`sum([1,2,3])` |
| **Day 12** | **this绑定** | 默认/隐式/显式/`new`绑定、箭头函数 | 手写`bind()` polyfill，不调用原生`bind` |
| **Day 13** | **回调与高阶函数** | 回调地狱、高阶函数、函数柯里化 | 将`fs.readFile`风格回调函数转为Promise风格 |
| **Day 14** | **周复盘** | 刷LeetCode "函数"相关题3道 | 用函数式编程重构一个旧项目 |

### **执行规则（专注力测试核心）**

### **每日仪式**
1. **早上6:30-7:00**：打开GitHub Issue，写`Day X: 今日目标`（不超过50字）
2. **编码时必须关门**：手机开飞行模式，用Pomodoro（25分钟编码+5分钟休息）
3. **提交代码必须写测试**：即使是1行函数，也要写`assert`或Jest测试
4. **晚上10:00-10:30**：写`README.md`回答3个问题：
   - 今天学了什么？
   - 明天学什么？
   - 有什么问题？如何改进

### **淘汰机制（专注力的试金石）**
- **断更1天**：第二天必须补双倍（6小时）
- **断更3天**：证明计划不适合你，立即降级到Plan A（系统补课）
- **解决方案想不出**：允许Google，但**必须手写3遍**并解释每行

### **学习资料清单**
- **MDN文档**：唯一权威来源，禁止看中文博客（除非MDN看不懂）
